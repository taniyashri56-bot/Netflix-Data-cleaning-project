# Netflix-dataCleaning-project
Data cleaning and preprocessing of a Netflix Movies and TV Shows dataset using Excel.
## Project Overview
The dataset is based on Netflix Movies and TV Shows and Contains missing values,inconsistent formatting,duplicates,and mixed data types.
## Data Cleaning Steps Performed
1. Opened dataset and verified column headers
2. Freeze header row and applied filters to it
3. Identified missing values using filters and conditional formatting
4. Removed duplicate records after creating a backup
5. Cleaned text data using TRIM, PROPER, and CLEAN functions
6. Standardized date and numeric formats
7. Fixed inconsistent categorical values
8. Created a separate cleaned dataset sheet
9. Exported cleaned data in Excel and CSV formats

## Final Outcome
A clean, structured dataset ready for analysis, demonstrating practical Excel data cleaning skills used in real-world data analyst roles.
